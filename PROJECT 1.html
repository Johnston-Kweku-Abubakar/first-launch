CSS

Your webpage or site needs to look good right, this is where CSS comes in. CSS is short for Cascading StyleSheet. This has to do with the appearance of some or all of the content or layout of the page, depending on developer's preferences. With just HTML, the website would not look pleasing to the eye. The site would then need some styling and this is where CSS comes in. The "style" tag is used to indicate CSS elements. The style tag appears inside the head tag. (<head>). The style element comes in the form;
<style>
h1 (to specify first heading) {
color: blue
}
